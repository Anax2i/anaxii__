.osk (extension)
===================

L'extension **.osk** est une archive contenant un [skin osu!](/wiki/Skinning). Tout comme le fichier [.osz][Osz Link], le fichier .osk est un fichier compressé .zip.

Contenu des fichiers .osk
-------------------------

Un fichier .osk contient plusieurs fichiers divers et variés, dans cette liste, nous allons montrer quelques types de fichier. Pour plus d'informations sur **tous** les fichiers, il est recommandé de jeter un œil à <FR:Skinning>

-   Un fichier [skin.ini](/wiki/Skinning/Skin.ini), contenant toutes les règles du skin, comme son nom, mais aussi des donnés plus complexes, celui-ci peut s'éditer très facilement avec des logiciels de base comme Bloc Notes (Windows) ou TextEdit (Mac).
-   Des fichiers images, principalement .png, changeant le côté visuel du skin
-   Des fichiers sons, avec l'extension .wav, visant à changer les "[Hitsounds](/wiki/Glossary)"

Lire un fichier .osk
--------------------

Lors d'un téléchargement de fichier .osk, il est très simple pour quiconque de l'installer, ou de le modifier.

-   osu! permet directement d'installer un skin, il suffit soit de glisser-déposer le fichier .osk dans le jeu, sois double-cliquer sur le fichier. osu! va vous installer le skin dans le dossier "Skins" et vous ouvrir une visualisation de ce skin, pour connaître les éléments de ce skin.
-   Avec des logiciels externes lisant les fichiers .zip, cela permettra de modifier le skin sans avoir besoin d'exporter celui-ci à chaque fois.
