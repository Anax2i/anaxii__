L'extension **.osb** est utilisée pour contenir les informations du [storyboard](FR:Storyboarding) comme le déplacement d'élément ou les effets qui lui sont attribués, mais les éléments ne sont **pas** contenus dans un fichier **.osb**.
Un fichier **.osb** est unique et ne se trouve qu'une fois par map.
