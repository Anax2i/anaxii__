# Référence de la clé de raccourci

Il existe de nombreux raccourcis dans osu ! qui peuvent être utilisés dans le jeu ou en mode veille sur n'importe quel écran. Ils sont énumérés ci-dessous avec leurs fonctions.

*Note : Cet article énumère les raccourcis clavier par défaut. Vous pouvez en modifier certains dans le menu [options](/wiki/Options).*

## Jeu complet

Ces raccourcis fonctionnent partout :

| Raccourci | Action |
| :-- | :-- |
| `Alt` + `F4` | Fermez le client du jeu (pas de messages). Si vous êtes dans le jeu, dans l'éditeur de carte de rythme ou dans osu!direct, retournez à la sélection des chansons ou au menu principal (pour osu!direct). |
| `Alt` + `Up/Down Arrow` | Réglez le volume principal, utilisez les flèches gauche et droite pour régler les effets et le volume de la musique. |
| `Ctrl` + `Alt` + `Shift` + `S` | Rechargez la peau actuelle. Si vous appuyez sur cette touche pendant le jeu ou à l'écran des résultats, cette opération sera exécutée après le retour à l'écran de sélection des chansons. |
| `Ctrl` + `F11` | Basculer le graphique des temps de trame. En appuyant sur `Ctrl` fera une pause. |
| `F7` | Passez en revue les limiteurs de fréquence d'images. |
| `F8` | Basculez la fenêtre de discussion. |
| `F9` | Basculer la fenêtre de chat étendue. |
| `F10` | Basculer tous les boutons de la souris (sauf la molette de la souris, qui est toujours activée). |
| `F12` | Faites une capture d'écran. Cliquez sur la boîte de dialogue pour ouvrir le dossier des captures d'écran. |
| `Shift` + `F12` | Faites une capture d'écran et téléchargez la vidéo sur le serveur pour la partager. La capture d'écran sera également sauvegardée localement. La capture d'écran s'ouvrira alors sur votre navigateur web. |
| `Insert` | Il s'agit de la [clé du patron](https://fr.wikipedia.org/wiki/Boss_key "La clé du patron sur Wikipédia"). Réduire le client du jeu dans la zone de notification. Cela permettra également de désactiver le jeu. |
| `Shift` + `F1` | Ouvrez [osu!status](https://twitter.com/osustatus "osu!status sur Twitter") sur votre navigateur web. (Uniquement lorsqu'une erreur de connexion apparaît). |
| `Ctrl` + `O` | Ouvrez la barre latérale des options. Impossible d'entrer dans l'assistant de décalage si vous n'êtes pas dans le menu principal. |

## Menu principal

| Raccourci | Action |
| :-- | :-- |
| `Alt` + `Enter` | Basculer en mode plein écran. |
| `Up/Down Arrow` | Réglez le volume principal, utilisez les flèches gauche et droite pour régler les effets et le volume de la musique. |
| `Esc` | Revenir en arrière ou annuler. |
| `P`, `E`, ou `O` | Ouvrez le menu de jeu. |
| `P` trois fois | Entrez dans Solo. |
| `P` deux fois, alors `M` | Entrez [Multi](/wiki/Multi). |
| `E` deux fois | Entrez en mode édition. |
| `O` | Ouvrez la barre latérale des options. |
| `Q` | Afficher l'invite de sortie. |
| `Esc` | Afficher l'invite de sortie, lorsque les options et le menu de lecture sont fermés. |
| `F` | Basculez le compteur du SPF. |

### Jukebox

| Raccourci | Action |
| :-- | :-- |
| `Z` ou `F1` | Jouez la chanson précédente. |
| `X` ou `F2` | Jouez la chanson. Si vous la jouez, recommencez la chanson. |
| `C` | Basculez la pause et jouez. |
| `F4` | Arrêtez de jouer (réinitialise l'heure actuelle). |
| `V` ou `F5` | Jouez la chanson suivante. |
| `J` ou `F6`| Ouvre le saut au menu. |
| `R` | Jouez une chanson au hasard. |

## Choix de la chanson

*Note : Toutes ces fonctions ne fonctionnent pas dans l'écran de sélection des chansons du mode Multi.*

| Raccourci | Action |
| :-- | :-- |
| `F1` | Ouvrez le [game modifiers](/wiki/Game_Modifiers) menu de sélection. |
| `F2` | Choisissez au hasard une carte de rythme. |
| `Shift` + `F2` | Retournez à la feuille de route précédemment sélectionnée au hasard. |
| `F3` | Ouvrez les options de la carte des temps. |
| `F5` | Retraiter les beatmaps (cela peut vous inciter à continuer). |
| `Shift` + `Delete` | Invitez le dialogue de suppression de la carte des temps. Il vous sera demandé si vous souhaitez supprimer le beatmapset (toutes les difficultés), le beatmap (la difficulté sélectionnée) ou tous les beatmaps visibles. |
| `Ctrl` + `1` | Passez le mode jeu en [osu!](/wiki/osu!). |
| `Ctrl` + `2` | Passez le mode jeu en [osu!taiko](/wiki/osu!taiko). |
| `Ctrl` + `3` | Passez le mode jeu en [osu!catch](/wiki/osu!catch). |
| `Ctrl` + `4` | Passez le mode jeu en [osu!mania](/wiki/osu!mania). |
| `Ctrl` + `Enter` | Jouez le beatmap sélectionné avec le [auto mod](/wiki/Auto). |
| `Ctrl` + `Shift` + `Enter` | Jouez le beatmap sélectionné avec le [cinema mod](/wiki/Cinema). |
| `Ctrl` + `A` | Basculer l'auto mod. |
| `Ctrl` + `F3` ou `Ctrl` + `+` | Augmentez la vitesse de défilement de osu!mania (en mode osu!mania). |
| `Ctrl` + `F4` ou `Ctrl` + `-` | Réduisez la vitesse de défilement de osu!mania (en mode osu!mania). |
| `Ctrl` + `Backspace` | Supprimez le dernier mot dans la barre de recherche. |
| `Up/Down Arrow` | Sélectionnez respectivement la prochaine carte de battement en haut/en bas. |
| `Left/Right Arrow` | Sélectionnez respectivement le prochain beatmapset up/down. |
| `Page Up/Down` | Faites défiler la page respectivement vers le haut et vers le bas. |
| `Enter` | Entrez la sélection actuelle (si la sélection se fait sur un groupe de beatmapset ou de collection) ou en train de jouer une difficulté (si la sélection se fait sur un beatmap). |
| `Shift` + `Left/Right Arrow` | Entrez le groupe suivant/précédent. |
| `Shift` + `Enter` | Élargir ou réduire le groupe sélectionné. |

## Mode de jeu

| Raccourci | Action |
| :-- | :-- |
| `+` ou `-` | Ajustez le décalage local (maintenez `Alt` pour changer la précision). |
| `Esc` | Pause du jeu, ou retour à la sélection des chansons si vous regardez une relecture ou si vous utilisez le mode Auto ou Cinéma. |
| `Space` | Passez au jeu. |
| `Tab` | Basculer le tableau d'affichage. |
| `Shift` + `Tab` | Masquage de l'interface de jeu par basculement. |
| `H` | Permet de masquer le tableau d'affichage et l'interface de rediffusion (lors d'une rediffusion ou d'un visionnage). |
| `F3` ou `F4` | Ajustez la vitesse pour osu!mania (peut être utilisé vers le début du jeu). |
| `Ctrl` + `R` ou `` ` `` | Doit être tenu brièvement. Réessayez rapidement la carte actuelle (sautez l'introduction lors de la réessai). |

## Mode lecture

*Avis : Vous devez utiliser le flux de diffusion de pointe pour les utiliser.*

| Raccourci | Action |
| :-- | :-- |
| `5` | Pause de la rediffusion. |
| `6` ou `0` | Reprenez ou réglez la vitesse de lecture à 1x. |
| `7` | Réglez la vitesse de lecture à 2x. |
| `8` | Réglez la vitesse de lecture sur 3x. |
| `9` | Réglez la vitesse de lecture sur 4x. |

## Éditeur de Beatmap

*Page principale : [Beatmap Editor](/wiki/Beatmap_Editor)*

### Généralités

Ces raccourcis fonctionnent n'importe où dans l'éditeur de beatmap :

| Raccourci | Action |
| :-- | :-- |
| `Z` | Passez à la première note ou au début de la chanson. |
| `V` | Passez à la dernière note ou à la fin de la chanson. |
| `X` | Jouez dès le début. |
| `C` ou `Space` | Pause ou reprise. |
| `Left Arrow` ou `Right Arrow` | Passez la chanson. |
| `Up Arrow` ou `Down Arrow` | Sauter entre les points de chronométrage. |
| `Shift` + `Left Arrow` ou `Shift` + `Right Arrow` | Parcourez la chanson à une vitesse de 4x. |
| `Ctrl` + `B` | Ajouter un signet à l'horodatage actuel. |
| `Ctrl` + `Shift` + `B` | Supprimez le signet dans l'horodatage actuel. |
| `Ctrl` + `Left/Right Arrow` | Aller au signet précédent / suivant. |
| `Ctrl` + `L` | Revenir au dernier état sauvegardé (partiel). Cela permet de recharger le `.osu` et `.osb` fichiers. |
| `Ctrl` + `Shift` + `L` | Revenir au dernier état sauvegardé (complet). Cela permet de tout recharger. |
| `Ctrl` + `O` | Ouvre la fenêtre de sélection de la difficulté. |
| `Ctrl` + `S` | Sauvez la feuille de route dans son état actuel. |
| `Ctrl` + `N` | Invite à retirer tous les objets touchés. |
| `Ctrl` + `Up/Down Arrow` | Augmenter/diminuer la vitesse de lecture de 25%.. |
| `Ctrl` + `Shift` + `Up/Down Arrow` | Augmentez ou diminuez la vitesse de lecture de 5 %. |
| `F1` | Passez à la [compose](/wiki/Compose) mod. |
| `F2` | Passez à la [design](/wiki/Design) mod. |
| `F3` | Passez à la [timing](/wiki/Timing) mod. |
| `F4` | Ouvrez le [song setup](/wiki/Song_Setup) dialogue. |
| `F5` | Entrer en mode test. |
| `F6` | Ouvrir le dialogue sur le calendrier et les points de contrôle. |
| `Ctrl` + `P` | Ajouter un point de chronométrage non hérité à l'horodatage actuel. S'il n'est pas en mode de chronométrage, cela ouvrira également le dialogue des points de chronométrage et de contrôle. |
| `Ctrl` + `Shift` + `P` | Ajouter un point de chronométrage hérité à l'horodatage actuel. Cela ouvrira également le dialogue des points de contrôle et de synchronisation. |
| `Ctrl` + `I` | Supprimez la section actuelle non héritée ou héritée. Cela ne fonctionnera pas sur le premier point de synchronisation non hérité. |

#### Souris

| :-- | :-- |

| `Shift` + `Mouse Wheel Up/Down` | Cherchez en avant/en arrière à 4x. |

| `Alt` + `Mouse Wheel Up/Down` | Ajustez le multiplicateur de l'accrochage de la distance en survolant le terrain de jeu. Ajuster le zoom de la ligne de temps lors du survol de la ligne de temps. |

| Raccourci | Action |
| :-- | :-- |
| `Alt` + `F4` ou `Esc` | Retour à la sélection des chansons. |
| `Ctrl` + `Z` | Annuler. |

#### Manipulation d'objets

| Raccourci | Action |
| :-- | :-- |
| `Q` | Basculer le nouveau combo. |
| `Ctrl` + `,` | Faites pivoter la sélection de 90 degrés dans le sens inverse des aiguilles d'une montre. |
| `Ctrl` + `.` | Faites pivoter la sélection de 90 degrés dans le sens des aiguilles d'une montre. |
| `Ctrl` + `H` | Retournez la sélection horizontalement. |
| `Ctrl` + `J` | Retournez la sélection verticalement. |
| `Ctrl` + `G` | Inverser la sélection. C'est-à-dire, déplacer le premier objet sélectionné vers le dernier objet sélectionné et ainsi de suite. |
| `Ctrl` + `Shift` + `R` | Faites pivoter les objets sélectionnés selon un angle spécifié (ouvre une boîte de dialogue). |
| `Ctrl` + `Shift` + `S` | Échelle des objets sélectionnés d'un montant spécifié (ouvre une boîte de dialogue). |
| `Ctrl` + `Shift` + `D` | Générer des polygones avec un nombre de points spécifié (ouvre un dialogue). |
| `Ctrl` + `Shift` + `F` | Convertissez le curseur sélectionné en un flux de cercle de frappe (ouvre une boîte de dialogue). |
| `Ctrl` + `Left/Right/Up/Down Arrow` | Poussez les notes sélectionnées dans le sens de la pression. (osu!standard, osu!taiko, osu!catch) |
| `Ctrl` + `Left/Right Arrow` | Déplacez les objets sélectionnés vers la gauche ou la droite d'une colonne. (osu!mania) |
| `Ctrl` + `Up/Down Arrow` | Déplacez les objets sélectionnés vers le haut ou vers le bas d'une coche dans le temps. (osu!mania) |
