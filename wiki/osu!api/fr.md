# osu!api

osu! possède une [API](https://fr.wikipedia.org/wiki/API) publique que des services tiers peuvent utiliser pour développer et soutenir la plateforme osu!. Avant, on ne pouvais l'utiliser que sur demande, mais tout le monde peut maintenant utiliser l'API tant qu'il a un compte osu!.

Vous pouvez trouver la documentation de l'API sur [le wiki d'osu-api](https://github.com/ppy/osu-api/wiki). La version actuelle de l'API utilise des clés privées pour authentifier les demandes. Vous pouvez demander une clé API sur la [page d'application](https://osu.ppy.sh/p/api) sur le site d'osu!. Notez que la documentation sur le wiki osu-api n'est peut-être pas complète ni étendu et que l'API elle-même n'est pas en cours de développement actif.

---

La version 2 de l'API osu! est en développement sur [osu-web](https://github.com/ppy/osu-web), et vise à être beaucoup plus robuste que l'original. La documentation (dont la mise à jour n'est pas garantie) peut être trouvée sur cette [page de référence](https://docs.ppy.sh).

L'authentification pour la nouvelle API suit le [protocole OAuth 2](https://oauth.net/2/). Les clients peuvent être gérés via la [page des paramètres de votre compte](https://osu.ppy.sh/home/account/edit).
