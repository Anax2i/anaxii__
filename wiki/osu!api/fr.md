# osu!api

osu! a une [API](https://fr.wikipedia.org/wiki/Interface_de_programmation) publique que les services tiers peuvent utiliser pour étendre et soutenir la plateforme osu ! Auparavant, elle était privée et ne pouvait être utilisée qu'à la demande, mais maintenant, tout le monde peut utiliser l'API tant qu'il possède un compte osu !

Vous pouvez trouver de la documentation sur l'API à l'adresse suivante : [osu-api](https://github.com/ppy/osu-api/wiki). La version actuelle de l'API utilise des clés privées pour authentifier les demandes. Vous pouvez demander une clé API à l'adresse suivante : [page de l'application](https://osu.ppy.sh/p/api) sur le site web d'osu!

Notez que la documentation sur le wiki osu!api peut ne pas être entièrement complète ou étendue, et l'API elle-même n'est pas en cours de développement actif. Une nouvelle API, beaucoup plus robuste, est en cours de développement à l'adresse [osu-web](https://github.com/ppy/osu-web).
