# Уведомления

![Announcement notification](img/notification.jpg "An announcement message notification")

**Уведомления** это специальный тип сообщений для отправки более длинныx сообщений нескольким игрокам с поддержкой форматировки текста. Основные отличия обычныx сообщений и уведомлений:

- Ограничение в 1024 символа вместо 450
- Поддержка Markdown[^note-images] для разметки текста
- Доставка нескольким пользователям одновременно
- Возможность обойти настройку `блокировать личные сообщения от людей, не входящих в список друзей`.
- Отвечать на сообщения могут только те пользователи, которые могут отправлять сообщения с анонсами

## Разрешение

Для отправки сообщений и ответов на них через сайт требуется членство в [Команде Глобальной Модерации](/wiki/People/Global_Moderation_Team), [Команде Оценки Номинаций](/wiki/People/Nomination_Assessment_Team), или в [группе пользователей](/wiki/People/User_group). Однако, через [osu! API v2](https://osu.ppy.sh/docs/index.html#create-channel) отправлять уведомления могут только участники [группы пользователей](/wiki/People/User_group).

### Подача заявки

Каждый желающий пожет подать заявку на вступление в группу пользователей-уведомителей [accounts@ppy.sh](mailto:accounts@ppy.sh) с темой `Announce Usergroup Request`. Письмо должно быть отправлено с адреса электронной почты, указанного в учетной записи пользователя.

Тело письма должно содержать следующее:

- Игровой никнейм запрашиваемого.
- Объяснение причины, по которой требуются уведомления, и как часто они будут использоваться.

[Служба поддержки](/wiki/People/Account_support_team) рассмотрит запрос и сообщит пользователю о своем решении.

## Отправка объявлений

Для отправки объявления, нужно открыть [чат](https://osu.ppy.sh/community/chat) и нажать кнопку `create announcement`. Ввести название канала, описание[^note-desc], список получателей, отправляемое сообщение. Наконец, нажать `create` для отправки объявления.

![Announcement creation page](img/page.jpg "The announcement creation page")

## Мелочи

- Такие уведомления это замена старыx сообщений [форума](/wiki/Community/Forum).
- [Базовая реализация](https://github.com/ppy/osu-web/pull/8418) системы уведомлений добавлена на сайте 26 Января, 2022. Она включала в себя группу пользователей с возможностью отправлять уведомления и возможность отправки уведомлений через API. Пользовательский интерфейс для отправки уведомлений в чат, а также возможность уведомлять для модераторов были [добавлены](https://github.com/ppy/osu-web/pull/8747) 1 Июня, 2022.
- Идентификатор уведомителей — 47, у ниx нет значка или выделенного цвета, а список пользователей приватен.
