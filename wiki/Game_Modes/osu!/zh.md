简介
====

什么是“应援团”？
----------------

[押忍！战斗！应援团](http://zh.wikipedia.org/wiki/%E6%8A%BC%E5%BF%8D%EF%BC%81%E6%88%B0%E9%AC%A5%EF%BC%81%E6%87%89%E6%8F%B4%E5%9C%98) 是任天堂DS上的一款节奏游戏。其游戏内容仅包含三种元素：在触屏上点击圆圈、在固定的路径上拖动小球以及高速转动转盘。所有的这些动

作都是跟随着日本流行歌曲的节奏进行的。在DS平台上的截图如下：

![](Ouendan.jpg "Ouendan.jpg")

游戏中，上屏表示主人公的努力过程，下屏作用是游戏操作，背景有应援团团员做的各种动作。每关的基本内容是关于一个在生活中遇到困难的人。

你将通过应援团的神奇力量，为他们加油助威，使他们走出困境。

如何进行游戏？
--------------

由于大部分的PC玩家并没有安装触屏等设备，因此本游戏可以使用普通的鼠标，或者你能想到的任何输入设备来进行操作。不同的游戏方法可以在

[游戏风格页面中找到](ZH:Play_Styles "wikilink")。游戏中有一些歌曲可能会挑战你的鼠标（或者你的手）的极限，但是游戏所有（ranked或approved）的

谱面都经过测试，（大部分）都能使用鼠标过关。

### 游戏基础

`教程谱面: `[`（预置于PC版osu!游戏中）原版教程谱面`](http://osu.ppy.sh/s/3756)`, `[`Ranked版本教程谱面`](http://osu.ppy.sh/s/19928)

这款游戏主要需要你的：

-   反应能力（及时点击出现的物件）
-   对歌曲节奏的掌握
-   对鼠标的控制能力（如果使用数位板则是运笔的能力了）
-   对于物件在屏幕上出现时间的估测能力
-   使用鼠标（手指、笔）高速转动的能力
-   耐力（特别是马拉松歌曲）
-   鼠标定位和读图能力
-   对于各种硬件的熟悉

这些是你在成为osu!神触之路上将会学习和掌握到的一些重要技巧。在你成为神触之前，你需要学习一些关于击打物件的基础知识。

下面是关于这些物件的一些基础知识。

#### 圆圈

<img src="Hitcircles.jpg" title="Hitcircles.jpg" alt="Hitcircles.jpg" width="400" height="300" />

-   图中的实心圆圈就是需要在相应时间点击的物件。
    -   实心圆圈外面会有一个大圆圈，当大圈正好收缩到小圈的白色边缘的那一时刻，就应当点击这个物件。
-   为了深入理解圆圈的击打规律，你需要做的不仅仅是用眼睛去判定，更重要的是需要了解这些物件是跟随的歌曲的节奏的，需要用耳朵来判断。

#### 滑条

<img src="Slider.jpg" title="Slider.jpg" alt="Slider.jpg" width="400" height="300" />

-   图中的物件就是一个滑条。
    -   首先，你要像点击一个圆圈一样点击在滑条的头部，但是此时并不要松手。
    -   此时一个小球将会出现，沿着滑条的轨迹移动。
    -   你必须保持鼠标点击/键盘按下（或者手指/笔尖触板）的状态，使得光标位于小球外围的橙色圆圈中，并跟着小球一起移动。
        -   小球的移动不受你的控制，所以你必须跟着它来移动。
        -   当你保持按住鼠标/键盘的状态下，橙色圆圈才会出现，否则它不可见。
    -   有时，比如图中的情况，当你到达滑条末端时还需要原路返回。
    -   滑条末端的箭头形状可以指示上一条所说的情况。

#### 转盘

<img src="Spinner.jpg" title="Spinner.jpg" alt="Spinner.jpg" width="400" height="300" />

-   图中的物件是一个转盘。
    -   当它出现时你需要按下鼠标/键盘上表示点击的键，并使用鼠标疯狂地转圈直到表示倒计时的蓝色圆圈完全收缩。
        -   转盘外面的条显示你还差多少通过这个转盘。
        -   转盘下面的小横条上显示当前的转速，表示当前的速度下每分钟可以转多少圈。
        -   如果你在转的时候不按下表示击打的键，转圈不会有任何用。
-   在DS上，这是毁屏幕的绝佳方法（尤其是高难度的曲子）。
-   在osu!中，转盘的难度进行了一些调整，不至于太难完成。
    -   可以达成转速的上限是477圈每分，这也是开启Auto模式后达到的速度。开启Spun Out模式时，速度有所降低，为287转每分。
    -   学着尝试去转小圈（像Auto所做的一样，但如果不习惯的话会把你的手玩坏）。
-   \[鼠标用户请看\]'''如果无论你怎么转，光标并不是在旋转而是随机乱跑，请在以下方法中选择：查一下鼠标的移动速度上限、在平而光滑的木制表

面上使用鼠标、换个好点的鼠标垫。以上方法都不好用时，请买一个新的XD'''

-   -   地摊鼠可能经常遇到这种问题，最好**由慢到快地转而不是突然高速旋转**保证鼠标能够适应。
    -   **建议使用光滑的木质表面或鼠标垫**。除非你的鼠标支持，否则最好不要在光滑的玻璃表面或坑洼的表面上使用。

### 游戏时界面

![](Hierophobicintro.jpg "Hierophobicintro.jpg")

这是osu!模式游戏时的界面。

-   左上角的长条是血条。
    -   生命值会按照固定的速度减少（取决于作图者的设定），但是你可以通过正确的击打物件来补充生命值。
    -   完全准确的击打（300判定或“激”判定）补充的生命值要多于不准确的击打（比如50判定）。
    -   完全未命中的击打会导致大量扣血。
-   血条右侧显示你的得分。
-   得分下面表示准确率。
-   准确率旁边的饼图表示整个歌曲的进度，可以在“[设定](ZH:Options#详细设定 "wikilink")”中的“进度条位置”选项中修改。
-   屏幕左下角的数字表示连击数。

得分计算
--------

本节说明了所有关于分数计算的方法，包括详细的计算公式。

### 准确率

| colspan=2|准确率=总击打得分数/（总击打数×300） |
|------------------------------------------------|
| 总击打得分数                                   |
| 总击打数                                       |

也就是说，每个miss计0分，50判定计50分，100判定计100分，300判定计300分。这些分数的总和除以假设所有判定都是300的分数总和即为准确率。

比如你所有的判定都是300，准确率就是100%，而所有判定都是100时，则得到33.33%的准确率。

### 分数

每个圆圈和滑条尾端的计算公式如下：

| colspan=2|分数=击打判定值+击打判定值×（连击加成×难度加成×Mod加成）/25 |
|-----------------------------------------------------------------------|
| 击打判定值                                                            |
| 连击加成                                                              |
| 难度加成                                                              |
| Mod加成                                                               |

另外，在每个滑条的起始、结束和折返端都会得到30分，每个滑条中间点得到10分，转圈每转一圈得到100分。此外，转盘转满后，每转一圈还会得

到1000分的额外奖励。

### 物件判定

**<u>圆圈</u>**

-   根据你击打的准确程度，你将取得300、100、50的得分判定。
-   如果你根本没有击打或者击打得太早，会得到miss判定。

**<u>滑条</u>**

-   滑条的判定不依赖于起始击打的准确程度。
-   滑条包含中间点，包括开头、结尾以及中间重复的小点。
-   如果命中所有的中间点，获得300判定；如果命中至少一半，获得100判定；只要命中一个以上，就可以获得50判定。
-   **过早地点击滑条**并不会导致miss，但是将会**把分数加成置为0**。
-   只有当一个中间点都没命中时，才会得到miss判定。

**<u>转盘</u>**

-   对于转盘，会根据你转的圈数与转盘的总长度相比较来判定你得到300、100或50的判定。
-   如果你在转盘结束之前未填满转盘能量条，则得到miss判定。

### 分数加成

下列每一项都会为分数加成系数加一：

-   成功击中圆圈
-   成功击中滑条头
-   成功击中滑条尾
-   成功命中滑条中间点
-   成功完成或完成一半转盘

以下每项都会将分数加成系数归零：

-   圆圈获得miss判定
-   过早击中滑条头
-   未击中滑条头或滑条中间点
-   未能充满转盘能量条的一半

以下情况不会影响分数加成系数：

-   未命中滑条尾（这将导致一个“喝”判定或100/50判定）
-   转盘的1000分奖励

### 评分

**<u>一般评分</u>**

-   SS = 100% 准确率
-   S = 超过90%的300判定, 少于1%的50判定并且没有miss判定
-   A = 超过80%的300判定并且无miss判定**或者**超过90%的300判定
-   B = 超过70%的300判定并且无miss判定**或者**超过80%的300判定
-   C = 超过60%的300判定
-   D = 其他情况

<u>**特殊评分**</u>

-   银SS (SSH) = 100%准确率并开启'[Hidden](ZH:hidden "wikilink")'和'[FlashLight](ZH:flashlight "wikilink")'两个mod的任意组合。
-   银S (SH) = 超过90%的300判定, 少于1%的50判定并且无miss判定并开启'[Hidden](ZH:hidden "wikilink")'和'[FlashLight](ZH:flashlight "wikilink")'两个mod的任意组合。

<u>**关于个人页面的“Ranks”数据**</u>

-   个人页面的Ranks区域显示的评分数由你在线打图的评分数量确定。
    -   包括你在线时所进行的**离线游戏**都**不计入**评分数。
-   **评分数与每个谱面难度是相关的，不是整个歌曲的**。
    -   如果谱面有3个难度，你分别都取得了SS，你将得到3个SS判定数而不是一个。

### 生命值

生命值的计算系统较为复杂所以不再详细解释。这些均与作图者自己的设定有关。

下列情况会获得生命值：

-   击中一个圆圈。（好的判定获得更多的生命值。同一颜色的combo中，最后一个圆圈获得更多的生命值。）
-   击中滑条头、尾或折返端。（滑条尾上，好的判定获得更多的生命值。）
-   旋转或完成转盘（好的判定获得更多的生命值）
-   完成转盘后继续旋转（获得的生命值取决于转速）

下列情况会损失生命值：

-   固定扣血
-   在圆圈或滑条上取得miss判定
-   未完成转盘
-   完成转盘后未继续转

下列情况不会改变生命值：

-   休息时间
-   完成转盘后继续旋转

常见问题
========

哪里能找到谱面？
----------------

*关于谱面的详细解释位于条目 [Beatmaps中](ZH:Beatmaps "wikilink")。*

你可以在主页上的 "[Beatmaps](http://osu.ppy.sh/p/beatmaplist)" 中进行搜索。

-   你需要登录你的osu!账户，服务器不允许未登录的用户从官方网站进行下载。

我可以关闭Kiai时段吗？
----------------------

不行。不像视频，它们是游戏的一部分，所以不能被关闭。
