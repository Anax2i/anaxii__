**플레이 방법**
---------------

### **기본**

여타 다른 건반게임들과 마찬가지로 노트가 내려오는 라인의 키를 타이밍에 맞춰 누르면 된다.

긴 모양의 롱노트는 해당 키를 계속해서 누르고 있다가 롱노트가 끝나는 시점에 떼주면 된다.

### **키 설정**

#### **입력 장치 탭을 통한 통합 키 설정**

-   Left to Right: IIDX, EZ2DJ 등과 같이 아케이드 건반게임 형태로 키를 배열하고 싶을때 선택.

**스코어링**
------------

이 문단에서 점수, 판정, 랭크와 관련된 전반적인 내용을 다룬다.

### **판정**

판정은 MAX300, 300, 200, 100, 50, Miss의 6가지가 있으며, 판정의 범위는 다음과 같다.

-   MAX300: 노트를 가장 완벽한 타이밍에 처리했을 경우. 롱노트의 처음과 끝을 모두 정확히 누르고 뗐을 경우.
-   300: 노트를 적절한 타이밍에 처리했을 경우. 롱노트의 처음과 끝을 비교적 정확히 누르고 뗐을 경우.

이 이하의 판정부터는 정확도가 감소한다.

-   200: 노트를 적정타이밍에 살짝 어긋나게 처리했을 경우, 롱노트를 조금 일찍/늦게 누르거나 뗐을 경우
-   100: 노트를 적정타이밍에 어긋나게 처리했을 경우, 롱노트를 늦게 눌렀을 경우.
-   50: 노트를 사실상 다른 타이밍에 처리했을 경우.
-   Miss: 노트를 누르지 못하고 놓쳤을 경우, 롱노트에 아무런 조작도 하지 않고 아예 놓친 경우.

### **랭크**

정확도에 따른 랭크 산출 기준은 다음과 같다.

-   SS: 모든 노트를 MAX 및 300점으로만 처리
-   S: 정확도 95% 초과
-   A: 정확도 90% 초과
-   B: 정확도 80% 초과
-   C: 정확도 70% 초과
-   D: 정확도 70% 이하

### **점수 계산**

다른 종목과 달리 1,000,000점 만점의 만점제를 사용한다.

노트 하나를 처리했을 때 얻는 점수는 기본 점수 + 추가 점수로 구성된다.

각각의 계산 방법은 다음과 같다.

-   기본 점수: (1000000 \* 모드 배율 \* 0.5 / 총 노트수) \* (판정값 / 320)
-   추가 점수: (1000000 \* 모드 배율 \* 0.5 / 총 노트수) \* (가중치 \* √(보너스) / 320)

**참고**: 각 점수 모두 소숫점이 있을 경우 버린다.

-   판정값 = MAX300 -&gt; Miss 순으로, 320,300,200,100,50,0
-   가중치 = MAX300 -&gt; Miss 순으로 32,32,16,8,4,0
-   보너스 = 바로 전 노트에서의 보너스 + 부가치 - 감점치 / 모드 배율 (**0 미만으로 내려갈 수 없고, 최대 수치는 100을 넘을 수 없다.**)
-   부가치 = MAX300의 경우 2, 300의 경우 1
-   감점치 = 200 -&gt; Miss 순으로 8,24,44,무한

예:

-   노트 수가 113콤보인 곡의 none 모드 플레이에서 첫 노트를 MAX300으로 처리했을 경우.

`* 획득 점수: `**`8848`**

`* 기본 점수: (1000000 * 1 * 0.5 / 113) * (200 / 320) = 2765.486725663717 = `**`2765`**
`* 추가 점수: (1000000 * 1 * 0.5 / 113) * (16 * √(`**`92`**`) / 320) = 2122.049346598548 = `**`2122`**
`  보너스 = 100 + 0 - 8 / 1 = `**`92`**

`* 기본 점수: `**`0`**
