Introduction
============

**osu!stream** est une version de osu! créée à la base spécialement pour les appareils tactiles. Elle est actuellement disponible sur les appareils Apple [iOS](http://en.wikipedia.org/wiki/IOS) uniquement, mais une version [Android](http://en.wikipedia.org/wiki/Android_%28operating_system%29) est planifiée. Cela introduit un nouveau type de gameplay comme les cercles que l'on doit tenir et les cercles connectés.

Cette application ne supporte pas les maps classiques d'osu!. Les maps disponibles sont créées spécialement pour la version portable d'osu! par une équipe de mappers qualifiés. Toutes les maps gratuites et payantes sont disponible dans le "Store".
