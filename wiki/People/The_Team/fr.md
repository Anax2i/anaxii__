## Groupes d'utilisateurs

Les groupes d'utilisateurs suivants sont composés des membres de la communauté osu ! qui aident à maintenir osu ! La plupart de ces groupes d'utilisateurs sont reconnaissables à leurs couleurs de forum, leurs couleurs de chat en jeu, leurs titres de profil et/ou leurs badges de profil.

| Nom | Description |
| :-- | :-- |
| [Beatmap Nominators](Beatmap_Nominators) | Des utilisateurs qui vont au-delà de l'appel pour assurer la qualification de vos cartes |
| [Developers](Developers) | Rendre le jeu génial en ajoutant de nouvelles fonctionnalités et en corrigeant les bugs |
| [Global Moderation Team](Global_Moderation_Team) | Surveiller les forums et les discussions dans le jeu |
| [Nomination Assessment Team](Nomination_Assessment_Team) | Gestion des nominateurs de la Beatmap |
| [osu! Alumni](osu!_Alumni) | Ceux qui sont connus pour leurs contributions et qui sont depuis passés à autre chose |
| [Project Loved Team](Project_Loved_Team) | Reconnaître les rythmes que la communauté aime le plus |
| [Support Team](Support_Team) | Aide / Demande d'assistance |
