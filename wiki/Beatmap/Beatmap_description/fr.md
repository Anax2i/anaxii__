# Beatmap description

Les **Descriptions de Beatmap** sont de petites zones sous la page d'une [beatmap](/wiki/Beatmap) qui donnent aux joueurs des informations sur celle-ci, en utilisant le [BBCode](/wiki/BBCode). La plupart des beatmaps [classées](/wiki/Beatmap/Category#ranked) utilisent au moins une image, un nombre (en fonction du nombre de beatmaps classées que le créateur a) et une section de difficulté.

## Utilisations et informations

*Remarque: pour classer une beatmap, il est recommandé aux créateurs de l'acompagner d'une bonne description, mais ce n'est pas obligatoire.*

Les descriptions des beatmaps populaires incluent souvent les éléments suivants :

- Une liste de toutes les difficultés créateur et de leurs créateurs
- Un avertissement "langage 18+" (si nécessaire)
- Des informations qui aideraient un joueur à décider s'il souhaite télécharger la beatmap ou non
- Une liste de modifications ou un journal des modifications ("changelog")

### Images

De nombreuses descriptions de beatmap contiennent au moins une image (souvent un [GIF](https://fr.wikipedia.org/wiki/GIF "Wikipedia") animé de quelquonque sorte), qui est utilisé pour illustrer le créateur, la chanson, et/ou la beatmap d'une façon stylée. Ils peuvent également remplacer un titre ou un en-tête pour ajouter du style et de la netteté. Pour savoir comment ajouter une image à l'aide du BBCode, consultez le [tutoriel de l'article du BBCode sur les images](/wiki/BBCode#image).

### Sections de difficulté

Une "section de difficulté" est une liste de toutes les difficultés des invités et de leurs créateurs situés dans une description de la beatmap. Ceci est principalement utilisé pour montrer les difficultés d'une beatmap. Les sections de difficulté ont généralement des images pour les icônes.

Les icônes de difficulté se trouvent dans l'article des [difficultés](/wiki/Beatmap/Difficulty).
