# Pattern

A **pattern** is a small set of [hit objects](/wiki/Gameplay/Hit_object) within a [beatmap](/wiki/Beatmap) arranged in a meaningful way.

## Pattern types

### common

- [Double](common/double)

### osu!

- [Burst](osu!/Bursts)
- [Jump](osu!/Jump)
- [Cross Screen Jump](osu!/csj)
- [Stream](osu!/Stream)
