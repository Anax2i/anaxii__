# Obtainable Mod Introduction medals

| Medal name | Requirement(s) |
| :-- | :-- |
| Finality | Clear any Qualified, Ranked, or Approved beatmap with the [SuddenDeath](/wiki/Game_modifier/Sudden_Death) mod enabled. |
| Perfectionist | Clear any Qualified, Ranked, or Approved beatmap with the [Perfect](wiki/Game_modifier/Perfect) mod enabled. |
| Rock Around The Clock | Clear any Qualified, Ranked, or Approved beatmap with the [HardRock](/wiki/Game_modifier/Hard_Rock) mod enabled. |
| Time And A Half | Clear any Qualified, Ranked, or Approved beatmap with the [DoubleTime](/wiki/Game_modifier/Double_Time) mod enabled. |
| Sweet Rave Party | Clear any Qualified, Ranked, or Approved beatmap with the [Nightcore](/wiki/Game_modifier/Nightcore) mod enabled. |
| Blindsight | Clear any Qualified, Ranked, or Approved beatmap with the [Hidden](/wiki/Game_modifier/Hidden) mod enabled. |
| Are You Afraid Of The Dark? | Clear any Qualified, Ranked, or Approved beatmap with the [Flashlight](/wiki/Game_modifier/Flashlight) mod enabled. |
| Dial It Right Back | Clear any Qualified, Ranked, or Approved beatmap with the [Easy](/wiki/Game_modifier/Easy) mod enabled. |
| Risk Averse | Clear any Qualified, Ranked, or Approved beatmap with the [NoFail](/wiki/Game_modifier/No_Fail) mod enabled. |
| Slowboat | Clear any Qualified, Ranked, or Approved beatmap with the [HalfTime](/wiki/Game_modifier/Half_Time) mod enabled. |
| Burned Out | Clear any Qualified, Ranked, or Approved beatmap with the [SpunOut](/wiki/Game_modifier/Spun_Out) mod enabled. This medal can only be achieved in the osu! game mode as the SpunOut mod is not present in other game modes. |
