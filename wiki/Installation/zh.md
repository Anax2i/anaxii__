# 安装

*你也可以看看: [安装/macOS](/wiki/Installation/macOS)* <!-- and [Installation/Linux](/wiki/Installation/Linux)* -->

此页面将大致告诉您如何安装osu! [osu!academy](/wiki/osu!academy)有一个[视频教程](https://www.youtube.com/watch?list=PLmWVQsxi34bMYwAawZtzuptfMmszUa_tl&v=cz522ZAs5aQ) 来教您如何在Windows上安装osu!

## 最低系统要求

- .NET framework 4+ (安装程序将会提醒你下载，或者你可以[从这里](https://www.microsoft.com/en-us/download/details.aspx?id=48130))下载。
- 一个像样的显卡

## 安装osu!

1. 去[下载页面](https://osu.ppy.sh/home/download) 下载安装程序。
2. 找到安装程序并运行。
3. 当安装程序运行时，你有10秒的时间确定osu!安装位置。
    - 如果要更改安装位置，请单击单词“ Hi！”下方的文本，并选择安装位置。
    - 当计时结束时，osu!将安装到该位置。
4. 当安装结束，osu!将会自动打开，osu!direct将会为你下载一些入门谱面。
5. osu!将会提示您登入或者注册
    - 如果你已经有一个osu!帐号了，请登录
    - 如果你需要注册，可以看看 [注册](/wiki/Registration).

## 添加谱面

*你也可以看看: [谱面](/wiki/Beatmaps) 和 [谱面集](/wiki/Beatmapsets)*

有两种下载方式来添加谱面，通过网页或使用 (需要 [osu!supporter](/wiki/osu!supporter)).

### 网页

1. 前往[谱面页面](https://osu.ppy.sh/beatmapsets) 然后登录。
2. 寻找你想玩的歌。
3. 点击下载按钮将谱面集下载到你的电脑。
4. 定位然后打开谱面集文件，文件后缀为`.osz`
    - 另外，你也可以将谱面集文件复制到osu!的`Songs`目录来添加。
5. osu!将会处理其他的事并加载谱面集
    - 如果因为一些未知原因，你没看见添加的谱面，在单人模式按`F5`来重新读取谱面。

### osu!direct

*注意:你需要一个osu!supporter标签和一个可用的网络连接来使用osu!direct。*

1. 你如果没有打开osu!，请启动它。
2. 寻找你想玩的歌。
3. 双击列表中的歌曲，或单击右侧的`download`进行下载。
    - 如果您不想下载谱面集附带的视频，请单击`DL NoVideo`。
4. 当osu!完成谱面集的下载，将会自动在单人模式中的[歌曲选择](/wiki/Song_Selection)中自动加载。
    - 如果因为一些未知原因，你没看见添加的谱面，在单人模式按`F5`来重新读取谱面。

## 添加皮肤

*你也可以看看: [制作皮肤](/wiki/Skinning)*

1. 前往[skinning论坛](https://osu.ppy.sh/community/forums/15)然后寻找你喜欢的皮肤。
2. 一旦你找到了一个皮肤，下载它。
3. 根据皮肤制作者打包皮肤的方式，您可能需要做不同的事情
    - 如果皮肤使用`.osk`:
        1. 直接打开或者拖进osu!
        2. osu!将会自动安装并为你自动选择为该皮肤。
    - 如果皮肤压缩为 (使用`.7z`，`.rar`，`.zip`，或其他):
        1. 解压。
        2. 将皮肤文件夹移动到osu!的`Skins`文件夹中。 
        3. 如果osu!已经打开，你可能需要重新启动来使用皮肤
        4. 在osu!中，前往[选项](/wiki/Options)并搜索`Skin`。
        5. 向下滚动至`Current skin:`选项并选择你下载的皮肤
