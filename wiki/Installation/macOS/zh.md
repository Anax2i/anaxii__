# 在 macOS 上安装

如果你是 macOS 用户，这个页面将简要地为你描述如何在你的 macOS 设备上运行 osu!。

## 最低系统要求

- 一台比较好的 macOS 电脑

## 安装 osu!

macOS 版本的 osu! 仍然是实验性的。游戏可能不是很稳定，或者根本无法启动。这个[视频教程（YouTube）](https://www.youtube.com/watch?v=3ay221bMHNo)将帮助你理解下面的安装步骤。

1. 前往非官方的[下载页面](https://osu.ppy.sh/community/forums/topics/682197)，下载 `osu!.app.zip` 并将其解压。
2. 你将看到 `osu!.app` 文件夹（或者直接是一个应用程序），右键单击它选择 `打开` 。
   - 直接左键双击可能不会启动 osu!，macOS 可能会认为这个应用是由未知开发者开发的。通过右键菜单选择 `打开` 就会解决这个问题。
3. osu! 安装程序将会自动运行。但是 osu! 的安装位置无法更改，它将会直接安装在 `osu!.app` 文件夹下。移动 `osu!.app` 的位置将会移动里面所有的内容，你可以右键单击选择 `显示包内容` 来查看文件夹里的内容。
4. 安装完成之后 osu! 就会立即启动，然后自动使用 osu!direct 下载几个新手谱面。
5. osu! 会提示你登录或注册。
   - 如果你已经有了账号，登录即可。
   - 如果你需要注册，参见[注册](/wiki/Registration)。
