# Паки

Паки карт это `.zip` файлы, содержащие карты. Название пака основано на том, что он содержит (например `Пак одобренных карт #7` будет содержать только [одобренных](/wiki/approved) карты).

Стоит отметить, что большинство паков:

- Примерно ~ 100 МБ (Taiko/osu!mania) или ~ 200 МБ (- / Одобренных)
- Содержут видео (если включено)

## Категории

Наборы карт делятся на 4 категории:

| Категория | Типы карт |
| -------- | ------------- |
| Стандарт | Ранкнутые или одобренные мапсеты |
| График | Набор песен раз в две недели |
| Тема | Общая тема песен |
| Исполнитель/Альбом | По Исполнителю/Альбому |

## Полное заполнение вашего списка песен

Это для тех, кто действительно хочет * полностью заполнить свой список песен * (а не просто загрузить свои любимые песни).

Имейте в виду, что вам понадобится достаточно много цифрового пространства для хранения этих битмапов. Как пакеты карт (и торренты) также загружаются вместе с видео и другими элементами скинов карт.

### Список паков карт
Загрузите все (-/Одобренные/Taiko/osu!mania) паки карт из категории **Стандарт** и больше нигде. Наборы (-/Одобренные/Taiko/osu!mania)  будут содержать все песни трех других категорий.

Вы можете найти и скачать паки карт тут: [osu! Список Паков Карт](https://osu.ppy.sh/beatmaps/packs).
