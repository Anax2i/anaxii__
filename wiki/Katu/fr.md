# Katu

*Voir aussi : [Geki](/wiki/Geki)*

Le terme **Katu (喝)**, *Katsu*, ou *Beat!*, est utilisé dans le [score](/wiki/Score) lorsqu'un joueur a terminé un comboset sans atteindre la plus grande [précision](/wiki/Gameplay/Accuracy) possible sur toutes les notes. Ce terme provient du jeu Nintendo DS Elite Beat Agents, qui est l'un des jeux sur lequel le [mode de jeu](/wiki/Game_mode) [osu!standard](/wiki/Game_mode/osu!) est basé.
