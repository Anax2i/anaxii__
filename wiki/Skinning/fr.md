# Skinning

Le skinning est l'une des principales caractéristiques d'osu! Il permet aux joueurs de dériver des éléments d’écorchage originaux
pour créer les leurs! Les skins peuvent être amusantes, thématiques, PRO ou presque impossibles à jouer.

Ensembles de skinning

*Voir également : [Liste des skins](/wiki/Ranking_Criteria/Skin_Set_List/).*

**Pour les beatmaps uniquement,** si votre skin beatmap contient un seul élément parmi les ensembles listés dans les critères de classement,
il doit contenir tous les autres éléments de cet ensemble de skinning. Cela ne concerne que les skins beatmap, mais d'autres
les skinners peuvent également vouloir considérer ces listes.
