Skinning osu!

Default numbers
------------

Numbers inside of the Hitcircles.
A path to the images can be set inside the skin.ini using the `HitCirclePrefix` command.

``default-X.png`` where X is a number 0 to 9

![](fs/hitcircleoverlay.png)
