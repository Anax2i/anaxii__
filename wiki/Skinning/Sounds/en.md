# Skinning Sounds

Notes:

-    Hitsounds should not have any more than a 5 millisecond delay (unless it has a special purpose)
-    `WAV`, `MP3`, and/or `OGG` are valid formats for sounds (but do note that `WAV` files are prioritized)

For this article, the "Audio" column links to the default sounds.
