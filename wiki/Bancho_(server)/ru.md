# Bancho (сервер IRC)

*У этого термина существуют и другие значения, см. [Bancho (значения)](/wiki/Disambiguation/Bancho).*

**Bancho** — сделанный собственноручно [Internet Relay Chat](/wiki/Internet_Relay_Chat) (*IRC*) сервер для osu!. При входе в аккаунт через клиент, osu! автоматически подключается к Bancho.
