# Настройки

![Меню настроек](img/options_basic.jpg "Меню настроек \(боковая панель слева\)")

В главном меню кликните `Options` или нажмите `O` (или `Ctrl` + `O`) чтобы открыть боковую панель с настройками osu!. Как только меню настроек будет открыто, вы можете начать вводить текст для поиска интересующей настройки. Поле с вашим запросом даст знать, если ничего не нашлось.

При изменении параметра, серая вертикальная полоса слева от него будет светиться желтым цветом. При нажатии на эту полосу значение будет сброшено обратно.

## Общее

![Иконка основных настроек](img/general.jpg "Иконка основных настроек")

В этой категории находятся настройки вашего аккаунта, языка, а так же какие обновления вы хотите получать.

### Вход

Если вы не вошли в аккаунт в игре, вам будет предложено войти.
Вы можете проигнорировать это, нажав `Esc`, кликнув кнопку «назад» или перейдя к экрану выбору песни.
Если вы не войдёте в аккаунт, игра воспримет вас как гостя, у вас не будет доступа к просмотру онлайн рекордов,
вы не сможете публиковать рекорды, а так же не сможете играть [вместе](/wiki/Multi) с другими людьми.

Если вы являетесь гостем, будет показано:

| Имя | Описание | Тип | Значение по умолчанию |
| :-- | :-- | :-- | :-- |
| `Логин` | Поле для ввода логина. | Текст | *(пусто)* |
| `Пароль` | Поле для ввода пароля. | Пароль | *(пусто)* |
| `Запомнить Логин` | Игра запомнит введённый логин. | Флаг | `Активен` |
| `Запомнить Пароль` | Игра запомнит введённый пароль. Так же вы будете оставаться в аккаунте при перезапуске игры. | Флаг | `Не активен` |
| `Войти` | Войти в osu! с помощью указанных данных. Нажав `Enter` вы так же можете войти. | Кнопка |  |
| `Зарегистрироваться` | Открывает страницу [регистрации аккаунта](/wiki/Registration). | Кнопка |  |


---

Если же вы вошли в аккаунт:

| Имя | Описание | Тип |
| :-- | :-- | :-- |
| `Логин: {username}` | Вызывает меню. | Кнопка |

---

Если вы кликните на `Логин: {username}`, появится меню:

| Имя | Описание | Тип |
| :-- | :-- | :-- |
| `1. Открыть профиль` | Откроет ваш профиль в браузере. | Кнопка |
| `2. Выйти` | Выйти из аккаунта. | Кнопка |
| `3. Поменять аватарку` | Поменять аватарку профиля (откроется в браузере). | Кнопка |
| `4. Отмена` | Закрыть это меню. Так же, можно нажать `Esc`. | Кнопка |

Вы так же можете открыть это меню, нажав на карточку профиля, где это возможно.

## Графика

![Иконка настроек графики](img/graphics.jpg "Иконка настроек графики")

В этой категории находятся настройки графики и частей интерфейса.

### Обработчик графики

| Имя | Описание | Тип | Значение по умолчанию |
| :-- | :-- | :-- | :-- |
| `Ограничение FPS` | Установить ограничение FPS. | Выпадающий список | `Optimal` |
| `Счётчик FPS` | Показывать счётчик FPS в правом нижнем углу экрана. | Флаг | `Не активен` |
| `Режим совместимости` | Использовать старый обработчик графики osu!, который совместим со старыми ОС. Это может помочь, если osu! неправильно работает. При активации перезапустит игру. | Флаг | `Не активен` |
| `Снизить кол-во потерянных кадров` | Может помочь при подёргивании и потере кадров. Немного снизит FPS. | Флаг | `Не активен` |
| `Обнаруживать снижения производительности` | Предупредит, если что-то мешает osu! работать быстро. | Флаг | `Активен` |

---

*Заметка: Когда игра находится не в активном окне, FPS ограничивается до 30 автоматически.*

*Предупреждение для игроков с ноутбуками: если вы выберете неограниченный FPS в игре, ваш ноутбук может перегреться!*

Если вы откроете список `Ограничение FPS`, появится выбор из следующих опций:

| Имя | Описание |
| :-- | :-- |
| `Вертикальная синхронизация` | Ограничивает FPS в игре до частоты обновления вашего монитора. |
| `Power Saving` | Ограничивает FPS в игре до половины от частоты обновления вашего экрана. |
| `Optimal` | Ограничивает FPS в игре до уровня в восемь раз выше чем частота обновления вашего монитора, но не превышает 960fps. |
| `Нет (в игре)` | Ограничивает FPS в игре до частоты обновления вашего монитора. |

- `Вертикальная синхронизация`: С точки зрения простого игрока, использование вертикальной синхронизации заставит игру ждать, пока каждый кадр отобразится на экране полностью.
    - Вы можете использовать это, если видите некоторые «разрывы» (когда нижняя часть игры отстает от верхней части игры).
    - Это может вызвать некоторое отставание или замедление, потому что игра должна ждать загрузки каждого кадра.
- `Нет (в игре)`: Неограниченный FPS устанавливается, если вы играете на карте.
    - Когда вы не играете, FPS ограничен до уровня в два раза выше чем частота обновления вашего монитора, но не превышает 240fps.
    - Использовать эту опцию **не рекомендовано**. Она может привести к прыжкам и падениям уровня FPS, что неблагоприятно влияет на игровой процесс.
    - Разница между использованием опции `Нет (в игре)` и `Optimal` почти не ощутима.

### Настройки экрана

| Имя | Описание | Тип | Значение по умолчанию |
| :-- | :-- | :-- | :-- |
| `Разрешение` | Устанавливает разрешение в игре. Доступные для выбора разрешения отличаются от монитора к монитору. | Выпадающий список |  |
| `Полноэкранный режим` | Запускает игру в полноэкранном режиме (обычно, это позволяет уменьшить задержку при отклике). | Флаг | `Активен` |
| `Render in native resolution` | Запускает игру в полноэкранном режиме, но остаётся в отведённом для неё пространстве, а свободное место по краям заполняет чёрными полосками (для тех, кто хочет играть в окне, но при этом уменьшить задержку при отклике). | Флаг | `Активен` |
| `По горизонтали` | Двигает игру по горизонтали в режиме `Render in native resolution`. | Ползунок | `0%` |
| `По вертикали` | Двигает игру по вертикали в режиме `Render in native resolution`.  | Ползунок | `0%` |

- Если `Полноэкранный режим` выключен, под его флагом может появится предупреждение `Оконный режим увеличивает задержку ввода!`.

| Имя | Описание | Тип | Значение по умолчанию |
| :-- | :-- | :-- | :-- |
| `Ползущие слайдеры` | Слайдеры разворачиваются из своей начальной точки. | Флаг | `Активен` |
| `Фоновое видео` | Воспроизводить фоновое видео во время игры или нет. | Флаг | `Активен` |
| `Сториборды` | Показывать сториборды в игре (при наличии). Они обычно содержат тексты песен или спецэффекты. | Флаг | `Активен` |
| `Комбо-всплески` | При достижении определённого комбо сбоку экрана появится картинка. | Флаг | `Активен` |
| `Вспышки от нот` | Каждая сыгранная нота оставляет вспышку после себя. | Флаг | `Активен` |
| `Эффекты шейдеров` | Включает эффекты шейдеров (вспышки, размытие, красный экран и т.д). Рекомендуется включить, но требует хорошей видеокарты и поддержки шейдеров 2.0. Нельзя включить, если шейдеры не поддерживаются видеокартой. | Флаг | `Не активен` |
| `Сглаживающий фильтр` | Включает сглаживание. Это автоматически включит `Эффекты шейдеров`. | Флаг | `Не активен` |
| `Формат скриншотов` | Устанавливает формат скриншотов: `JPEG (малый размер)` или `PNG (качество)`. | Выпадающий список | `JPEG (малый размер)` |

### Главное меню

| Имя | Описание | Тип | Значение по умолчанию |
| :-- | :-- | :-- | :-- |
| `Включить снег` | В главном меню с небес будут падать маленькие иконки текущего мода игры. (принудительно включено во время зимы). | Флаг | `Не активен` |
| `Параллакс` | Включает сдвиг фона в зависимости от положения курсора. | Флаг | `Активен` |
| `Показывать советы` | В главном меню будут отображаться различные полезные советы. | Флаг | `Активен` |
| `Включить голосовое приветствие` | osu! будет приветствовать вас при входе и прощаться при выходе. | Флаг | `Активен` |
| `Использовать музыкальную тему osu!` | При запуске вместо случайной песни будет воспроизводиться тема osu!. | Флаг | `Активен` |
| `Seasonal backgrounds` | Использовать пользовательские арты в качестве фона главного меню (и в качестве фона карт, у которых нет собственного фона). Изображения будут меняться с переключением песен. | Выпадающий список | `Sometimes` |

---

Если вы откроете список `Seasonal backgrounds`, появится выбор из следующих опций:

| Имя | Описание |
| :-- | :-- |
| `Sometimes` | Вы будете видеть сезонные фоны в течение каждого сезона. Команда osu!dev выберет, когда они будут удалены и заменены простыми фонами. |
| `Never` | Вы никогда не будете видеть сезонные фоны. Никакого аниму! |
| `Always` | Вы всегда будете видеть текущие сезонные фоны, вне зависимости от решения команды osu!dev по их замене. |

## Игра

### Общее

| Имя | Описание | Тип | Значение по умолчанию |
| :-- | :-- | :-- | :-- |
| `Затемнение фона` | Регулирует уровень затемнения фона во время игры (может быть установлен на каждой карте отдельно). | Ползунок | `80%` |

| `Изменять скорость прокрутки в osu!mania в зависимости от BPM карты` | Быстрые карты будут прокручиваться быстрее. | Флаг | `Не активен` |
| `Своя скорость для каждой карты osu!mania` | Для каждой карты osu!mania будет запоминаться скорость её прокрутки. | Флаг | `Не активен` |

---

Если вы откроете список `Прогресс`, появится выбор из следующих опций:

| Имя | Описание |
| :-- | :-- |
| `Справа вверху (круг)` | Использовать круговую диаграмму, чтобы показать оставшееся время до завершения. Располагается слева от точности. |
| `Справа вверху (полоса)` | Использовать короткую полоску для отображения оставшегося времени. Она располагается под точностью. |
| `Справа внизу` | Использовать короткую полоску для отображения оставшегося времени, находящиеся в правом нижнем углу. |
| `Внизу (полоса)` | Использовать короткую полоску внизу экрана для отображения оставшегося времени. |

---

Если вы откроете список `Шкала точности`, появится выбор из следующих опций:

| Имя | Описание |
| :-- | :-- |
| `Отключена` | Не показывать точность. |
| `Цветовая` | Использовать цветные блоки для отображения точности. (режим osu!catch по умолчанию использует `Отклонение от нуля`.) |
| `Отклонение от нуля` | Использовать счетчик для отображения точности. Это показывает, ударил ли игрок слишком рано или слишком поздно. |

### Список песен

| Имя | Описание | Тип | Значение по умолчанию |
| :-- | :-- | :-- | :-- |

### Отладка

#### Версия сборки

Посмотрите, какую версию сборки в данный момент имеет игровой клиент и какие обновления сборки получает игровой клиент. Нажав на эту кнопку, вы перейдете к заметкам о выпуске в браузере.

Версия сборки состоит из следующих частей:

```
b{YYYY}{MM}{DD}.{revision}{type}
```

- `{YYYY}` год во время сборки
- `{MM}` месяц во время сборки
- `{DD}` день во время сборки
- `{revision}` номер версии сборки
    - Если номер версии сборки отсутствует, всё после точки будет удалено.
- `{type}` тип сборки
    - Если тип сборки отсутствует, значит он — `Стабильная`.

## Мелочи

- Если вы вводите имя пользователя, но оставляете текстовое поле пароля пустым, osu! будет использовать это имя при сохранении рекордов локально.
- Открытие боковой панели с настройками автоматически вызывает функцию `osu! is-up-to-date!`, которая проверяет наличие новых обновлений.
- Настройка `Seasonal backgrounds` была добавлена после того как комьюнити понравилась такая инициатива.
    - Узнать больше о [этих изменениях](https://osu.ppy.sh/community/forums/topics/606931)

### История

- Старый экран опций был настоящим экраном с вкладками, кнопками и тёмно-бледно-голубым фоном.
- Старый экран настроек содержал в себе ещё один, дополнительный экран, использующийся для предпросмотра скина.
    - После того как экран параметров был перемещен на боковую панель, этот экран все еще был доступен, с помощью открытия файла скина.
    - Доступ к этому был удален после того, как появились предварительные просмотры скина и функции live preview.
