# osu! Philippines Nationals 2023 (osu!standard)

![PHNS2023 banner](img/banner.png)

The **osu! Philippines Nationals 2023 (osu!standard)** (***PHNS 2023***) is a one-on-one, osu!standard tournament made for osu! players based in the Philippines or of Filipino Heritage.

## Tournament schedule

| Event | Timestamp |
| --: | :-- |
| Registration phase | 2022-11-28/2022-12-17 |
| Qualifiers | 2023-01-06/2023-01-08 |
| Round of 32 | 2023-01-13/2023-06-15 |
| Round of 16 | 2023-01-20/2023-01-22 |
| Quarterfinals | 2023-01-27/2023-06-29 |
| Semifinals | 2023-02-03/2023-06-05 |
| Finals | 2023-02-10/2023-02-12 |
| Grand Finals | 2023-02-17/2023-02-19 |

## Prizes

| Placing | Prizes |
| :-: | :-- |
| ![Gold crown](/wiki/shared/crown-gold.png "1st place") | Unique profile badge, osu! merch, 4 months osu!supporter |
| ![Silver crown](/wiki/shared/crown-silver.png "2nd place") | 2 months osu!supporter |
| ![Bronze crown](/wiki/shared/crown-bronze.png "3rd place") | 1 month osu!supporter |
