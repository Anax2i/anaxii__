# osu! Philippines Nationals 2023 (osu!catch)

![PHNC2023 banner](img/banner.png)

The **osu! Philippines Nationals 2023 (osu!catch)** (***PHNC 2023***) is a one-on-one, osu!catch tournament made for osu! players based in the Philippines or of Filipino Heritage.

## Tournament schedule

| Event | Timestamp |
| --: | :-- |
| Registration phase | 2022-02-22/2022-03-11 |
| Qualifiers | 2022-04-02/2022-04-03 |
| Round of 16 | 2022-04-09/2022-04-10 |
| Quarterfinals | 2022-04-16/2022-04-17 |
| Semifinals | 2022-04-23/2022-04-24 |
| Finals | 2022-04-30/2022-05-01 |
| Grandfinals | 2022-05-07/2022-05-08 |

## Prizes

| Placing | Prizes |
| :-: | :-- |
| ![Gold crown](/wiki/shared/crown-gold.png "1st place") | Unique profile badge, 3,100 Philippine Pesos |
| ![Silver crown](/wiki/shared/crown-silver.png "2nd place") | 1,550 Philippine Pesos |
| ![Bronze crown](/wiki/shared/crown-bronze.png "3rd place") | 775 Philippine Pesos |
